Requirements

Please check the requirements before installing the script.

Server Requirements#

  • PHP Version ^7.3|^8.0
  • Dom PHP Extension
  • Pdo PHP Extension
  • Pdo_mysql PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • JSON PHP Extension
  • Fileinfo PHP Extension

Folder Permissions#

  • storage/framework/ 775
  • storage/logs/ 775
  • bootstrap/cache/ 775
